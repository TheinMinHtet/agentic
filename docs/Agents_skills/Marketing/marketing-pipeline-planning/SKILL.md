## Overview
This skill focuses on architecting the journey a customer takes from initial discovery to conversion. It emphasizes clarity in stages and the triggers needed to move a prospect forward.

## Usage
Use this to design the logical flow of your marketing engine before launching specific campaigns.

## Steps
1.  **Stage Definition**: Clearly define stages (e.g., Awareness, Interest, Consideration, Intent, Purchase).
2.  **Conversion Triggers**: Define what specific action moves a user from one stage to the next.
3.  **Lead Scoring**: Assign values to interactions (e.g., email open, whitepaper download, pricing page visit).
4.  **Channel Mapping**: Assign specific channels to each funnel stage.
5.  **KPI Setup**: Define success metrics for each transition point.
