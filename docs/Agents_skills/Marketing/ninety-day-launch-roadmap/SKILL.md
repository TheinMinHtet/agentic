## Overview
This skill takes high-level goals and breaks them into a sequenced 90-day execution plan. It focuses on momentum, accountability, and stage-gate reviews.

## Usage
Use this for planning a major launch, ensuring all team members are aligned on what happens when.

## Steps
1.  **Phase Breakdown**:
    *   **Days 1-30 (Setup/Pre-launch)**: Foundation, assets, tech stack, internal alignment.
    *   **Days 31-60 (Execution/Launch)**: Campaign rollout, user acquisition, real-time optimization.
    *   **Days 61-90 (Growth/Refinement)**: Retention strategies, data analysis, scaling successful channels.
2.  **Milestone Mapping**: Define "Go/No-Go" checkpoints at the end of each phase.
3.  **Deliverable Scheduling**: Map tasks to specific weeks.
4.  **Resource Allocation**: Assign owners to key workstreams.
